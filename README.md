## Features
- AI-powered symptom checker  
- Suggests possible diagnoses or next steps  
- Generates random helpful questions from the knowledge base  
- Tracks token usage and estimated costs  
- Saves all chat logs for review  

---

## Tech Stack
- **Python** (3.13)  
- **Flask** for backend  
- **HTML/CSS/JS** for frontend  
- **LangChain** + **FAISS** for knowledge retrieval  
- **OpenAI GPT-4** for AI responses  

---

## How It Works
1. You enter your symptoms in the chat.  
2. The system retrieves relevant information from its knowledge base.  
3. The AI generates a response, including recommendations.  
4. The response and token usage are logged automatically.
